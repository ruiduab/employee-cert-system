## 测试说明

本测试集用于验证乐享AI对话系统的**意图识别能力**：当用户输入不同query时，系统能否正确判断并触发对应的认证流程。

- **职场人群认证**：面向个人在职身份，触发工具链为【职场人群认证状态查询】→【职场人群认证】
- **企业认证（企业会员认证）**：面向企业采购/企业会员场景，触发企业会员认证流程
- **不触发认证**：属于普通产品咨询或无关意图，AI直接回答即可

## 一、触发"职场人群认证"流程的 Query

## 三、不应触发认证的 Query（边界 Case）

以下 query 属于普通产品咨询、售后问题或无关话题，AI 直接回答即可，**不应触发任何认证流程**。

### 3.1 普通产品咨询

| # | Query | 预期行为 |
|---|-------|---------|
| 1 | ThinkPad X1 Carbon多少钱 | 直接回答产品价格 |
| 2 | 联想有哪些轻薄本推荐 | 直接推荐产品 |
| 3 | ThinkPad和IdeaPad有什么区别 | 直接回答产品差异 |
| 4 | 16GB内存够用吗 | 直接回答配置问题 |
| 5 | 联想笔记本哪个系列最好 | 直接回答产品问题 |

| 7 | 联想官网现在有什么活动 | 活动咨询，直接回答 |

### 3.4 模糊/低置信度场景（需注意）

以下 query 含义模糊，模型需要正确判断，**不应误触发**认证流程：

| # | Query | 说明 | 预期行为 |
|---|-------|------|---------|
| 17 | 认证 | 极短且无上下文 | 应先询问用户意图，或调用两个状态查询工具根据用户现有状态引导 |
| 18 | 我想认证 | 无明确认证类型 | 根据用户当前身份状态引导对应认证（已是企业会员则优先职场认证） |
| 19 | 有没有优惠 | 无具体场景 | 直接询问用户场景（个人/企业），不直接触发认证 |
| 20 | 我是员工 | 仅声明身份，无意图 | 不触发认证，引导用户进一步表达需求 |
| 21 | 公司有什么福利 | 不明确是否针对联想员工 | 不触发认证，询问用户具体需求 |

## 四、意图区分对照表

| 用户说的关键词/场景 | 应触发流程 | 不应触发 |
|-----------------|----------|---------|
| 职工认证、职场认证、在职认证 | 职场人群认证 | - |
| 员工优惠、职工专属价、在职员工打折 | 职场人群认证 | 企业认证 |
| 我是[职业]（律师/程序员/设计师…）+购机需求 | 职场人群认证 | - |
| 企业采购、批量购买、公司统一采购 | 企业认证 | 职场人群认证 |
| 企业会员、企业账号、企业优惠 | 企业认证 | 职场人群认证 |
| 升级钻石会员、企业会员升级 | 企业认证（钻石升级） | 职场人群认证 |
| 在职员工买电脑有优惠吗（模糊） | 职场人群认证（优先） | - |
| 普通产品价格、配置、售后 | 不触发认证 | - |
| 单独说"认证"、"我想认证" | 调用状态查询判断后引导 | 直接触发任一认证 |
